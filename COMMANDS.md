# Danh sach lenh Radio Bot

Prefix mac dinh: `!`

Co the doi prefix bang bien `PREFIX` trong file `.env`.

## Slash Commands

| Lenh | Mo ta | Vi du |
| --- | --- | --- |
| `/play query:<link>` | Phat truc tiep link YouTube, SoundCloud hoac Spotify | `/play query:https://www.youtube.com/watch?v=...` |
| `/play query:<ten bai>` | Hoi ban chon YouTube hoac SoundCloud roi moi tim | `/play query:Em cua ngay hom qua` |
| `/skip` | Bo qua bai dang phat | `/skip` |
| `/pause` | Tam dung bai dang phat | `/pause` |
| `/resume` | Tiep tuc phat bai da tam dung | `/resume` |
| `/stop` | Dung nhac, xoa hang doi va roi kenh voice | `/stop` |
| `/list` | Xem hang doi hien tai | `/list` |
| `/nowplaying` | Xem bai dang phat | `/nowplaying` |
| `/help` | Hien thi huong dan trong Discord | `/help` |

## Prefix Commands

| Lenh | Mo ta | Vi du |
| --- | --- | --- |
| `!p <link>` | Phat truc tiep link YouTube, SoundCloud hoac Spotify | `!p https://soundcloud.com/...` |
| `!p <ten bai>` | Hoi ban chon YouTube hoac SoundCloud roi moi tim | `!p Em cua ngay hom qua` |
| `!skip` | Bo qua bai dang phat | `!skip` |
| `!s` | Alias cua `!skip` | `!s` |
| `!pause` | Tam dung bai dang phat | `!pause` |
| `!resume` | Tiep tuc phat bai da tam dung | `!resume` |
| `!stop` | Dung nhac, xoa hang doi va roi kenh voice | `!stop` |
| `!leave` | Alias cua `!stop` | `!leave` |
| `!list` | Xem hang doi hien tai | `!list` |
| `!l` | Alias cua `!list` | `!l` |
| `!nowplaying` | Xem bai dang phat | `!nowplaying` |
| `!np` | Alias cua `!nowplaying` | `!np` |
| `!help` | Hien thi huong dan trong Discord | `!help` |
| `!h` | Alias cua `!help` | `!h` |

## Link duoc ho tro

- YouTube video
- YouTube playlist
- SoundCloud track
- Spotify track, playlist, album

Nguoi dung can vao voice channel truoc khi dung lenh phat nhac.
