# Privacy Policy

Last updated: June 1, 2026

This Privacy Policy explains what data the Discord bot collects and how it is used.

## 1. Data We Collect

The bot may temporarily process Discord user IDs, server IDs, channel IDs, voice channel information, commands, and music search queries in order to provide bot features.

## 2. Data Storage

The bot does not sell user data. Some data may be stored temporarily for features such as queues, settings, or playback state.

## 3. How Data Is Used

Data is used only to operate the bot, respond to commands, manage music playback, and improve reliability.

## 4. Data Sharing

We do not sell or share user data with advertisers. Data may be processed by third-party services needed for bot functionality.

## 5. Data Deletion

Server owners or users may request data deletion by contacting the bot owner.
